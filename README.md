# Rest API Tutorial

## Prerequisite:
1. Python3 (https://www.python.org/downloads/)
2. Pip (https://pip.pypa.io/en/stable/)
3. Python packages: flask,request (can be installed using pip)

## Server
URL: https://localhost:5000/user
Methods supported: GET, POST, PUT, DELETE

### Run server
python server.py

## Client
To test client

### Run Client
